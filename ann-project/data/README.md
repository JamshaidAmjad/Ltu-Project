# Data

Do **not** commit actual datasets to this repository.

## How to Download the Datasets

1. Download the required dataset(s) from the source provided by your instructor or project brief.
2. Place the downloaded files into this `data/` directory.
3. This directory is listed in `.gitignore` (except this README) so your data files will not be tracked by git.

## Expected Files

Place your dataset files here before running any scripts. For example:

```
data/
├── train/
├── test/
└── ...
```
